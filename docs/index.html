Bookmarklet-Installation für "Schulnetz-Noten-Copy-Paste"

Ein Bookmarklet ist ein spezielles Lesezeichen, das JavaScript-Code enthält. Folgen Sie diesen einfachen Schritten, um das Bookmarklet "Schulnetz-Noten-Copy-Paste" zu installieren:

Installation

  1. Stellen Sie sicher, dass Ihre Lesezeichen-Leiste sichtbar ist:
    • Bei den meisten Browsern kann sie durch Drücken von Ctrl+Shift+B (Windows/Linux) oder Command+Shift+B (Mac) eingeblendet werden.
  2. Fügen Sie das Bookmarklet hinzu:
    • Ziehen Sie den folgenden Link per Drag-and-Drop in Ihre Lesezeichen-Leiste: Schulnetz-Noten-Copy-Paste

Verwendung

Um das Bookmarklet auszuführen, klicken Sie einfach auf das Lesezeichen "Schulnetz-Noten-Copy-Paste" in Ihrer Lesezeichen-Leiste, wenn Sie auf einer Webseite sind.

Hinweis

Einige Webseiten können die Ausführung von Bookmarklets blockieren. Wenn das Bookmarklet auf einer bestimmten Seite nicht funktioniert, versuchen Sie es auf einer anderen Webseite.
